## Steps to run the app
1. Generate a [NewsAPI API key](https://newsapi.org/register) and store it in a file inside the root folder

    ``` swift
   let NEWS_API_KEY = // YOUR_API_KEY
    ```
3. Build the app (Minimum Deployment: iOS 17)
